# Скрипты для управления платформой

Коллекция bash-скриптов для упрощения развертывания и управления платформой.

## Доступные скрипты

- `deploy.sh` - Развертывание платформы
- `backup.sh` - Резервное копирование данных
- `update.sh` - Обновление сервисов
- `monitor.sh` - Быстрый просмотр статуса
- `scale.sh` - Масштабирование сервисов

## Использование

```bash
# Сделать скрипты исполняемыми
chmod +x scripts/*.sh

# Запустить скрипт
./scripts/deploy.sh
```
