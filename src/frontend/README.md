# Платформа онлайн-курсов - Frontend

Клиентское React приложение для платформы онлайн-обучения.

## Функциональность

- Просмотр каталога курсов
- Регистрация и аутентификация пользователей
- Прохождение курсов
- Отслеживание прогресса
- Личный кабинет

## Технологии

- React 18
- React Router
- Axios
- Material-UI

## Разработка

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```
